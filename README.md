# Hakeem Chichi Pharmacy — POC Website

A modern, full-featured e-commerce POC website for Hakeem Chichi Pharmacy, Surat.

## Files
- `index.html` — Main website (all pages)
- `style.css` — Styling (dark green luxury aesthetic)
- `script.js` — Product data, cart, checkout, filtering

## Features
- 🛒 Full e-commerce: Browse, Add to Cart, Checkout
- 18 products across 8 categories
- Search, filter by category, price & rating
- Product detail pages
- About page with full legacy timeline
- 2 store locations (Ranitalao + Parle Point)
- Contact form
- Mobile responsive

## 📱 How to Deploy from Mobile Phone

### Step 1: Create a GitHub Account (if you don't have one)
1. Go to **github.com** in your mobile browser
2. Tap **Sign Up** and create a free account

### Step 2: Create a New Repository
1. After logging in, tap the **+** button (top right)
2. Select **New repository**
3. Repository name: `hakeemchichi-website` (or any name you like)
4. Set to **Public**
5. ✅ Check **Add a README file**
6. Tap **Create repository**

### Step 3: Upload the Files
1. In your new repository, tap **Add file → Upload files**
2. Tap **Choose your files** and select:
   - `index.html`
   - `style.css`
   - `script.js`
3. Scroll down, add commit message: "Add Hakeem Chichi POC website"
4. Tap **Commit changes**

### Step 4: Enable GitHub Pages (Free Hosting!)
1. In your repository, tap **Settings** (tab at top)
2. Scroll down to **Pages** section (left sidebar)
3. Under **Source**, select **Deploy from a branch**
4. Branch: **main**, folder: **/ (root)**
5. Tap **Save**

### Step 5: Access Your Live Website
- After 1–2 minutes, your website will be live at:
  `https://YOUR-USERNAME.github.io/hakeemchichi-website/`
- GitHub will show you the URL in the Pages settings

### 💡 Tips
- Any changes you upload will auto-deploy in ~1 minute
- Share the link with your friend to show the POC!
- For a custom domain (hakeemchichi.com), you can set it in Pages settings
